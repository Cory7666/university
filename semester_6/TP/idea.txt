Реализовать сервис по предоставлению VPN-конфигураций.
Сервис будет реализован в виде чат-бота для некоторого мессенджера или в виде веб-сайта.
Основной функционал: предоставление актуальных на данный момент конфигураций, сортировка по регионам, по показателю сетевых задержек.
Для аутентифицированных пользователей предоставляется дополнительный функционал: сортировка конфигураций по популярности среди пользователей и их оценкам, автоматизированное уведомление об устаревании конфигурации.
